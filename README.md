# Panorama

Portfolio platform for creative professionals. Built with Next.js 15, React 19, TypeScript, and Tailwind CSS.

## Structure

```
apps/web          Next.js frontend
packages/schema   Shared Zod schemas (Profile, Project, blocks)
```

## Getting started

```bash
npm install
npm run dev     # starts apps/web on localhost:3000
```

## Build

```bash
npm run build
```
